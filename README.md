# The Daily Grind - Lab 1

This is a basic Express.js server for a local coffee shop website.

## Project Structure

- `server.js` — Express server with routes
- `public/index.html` — Homepage
- `public/contact.html` — Contact page
- `public/css/style.css` — Optional styling
- `.gitignore` — Node modules ignored
- `package.json` — Project metadata and dependencies

## How to Run

1. Install dependencies: `npm install`
2. Start server: `node server.js`
3. Open browser:
   - Homepage: [http://localhost:3000](http://localhost:3000)
   - Contact: [http://localhost:3000/contact](http://localhost:3000/contact)
